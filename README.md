## minimal pokemon league web app made with PokeApi

### Running The Application
First, install all the dependencies,

```
npm i
# or
yarn
```
Then, run the development server:

```
npm run dev
# or
yarn dev
```

Open http://localhost:3000 with your browser to see the result.
